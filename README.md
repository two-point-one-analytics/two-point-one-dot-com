# Two Point One Analytics — Website

Static single-page website for **Two Point One Analytics**, a consulting business owned by Curtis Robinson.

## Purpose

Marketing site for an analytics consulting practice. Migrating from Squarespace to a self-hosted static site on Cloudflare Workers.

## Business Overview

- **Tagline**: "Bringing clarity to your online business"
- **Founder**: Curtis Robinson — 18+ years delivering analytics solutions and business insights
- **Services**:
  - **Analytics Implementation** — web analytics, tag management, data warehouse setup
  - **Report Creation** — site activity/attribution reports, ecommerce dashboards
  - **Data Analysis** — user behavior trends, marketing ROAS, churn/LTV, test design
- **Tool Expertise**: Google Analytics, GTM, Fivetran, Domo, Tableau
- **Notable Clients**: Acronis, AWWA, Casio, Gen Digital, Guatemala (gov), Hiya, JES, Norton, Ottawa Tourism, Ripe, skuIQ, Subrosa, Tenable, Tiki Brand

## Tech Stack

- Pure HTML/CSS/JS (no framework)
- Hosted via Cloudflare Workers (`wrangler.toml`)
- Domain: www.2point1analytics.com

## Project Structure

```
index.html          — Single-page site
images/             — Logo, hero, headshot, client logos, tool logos, footer bg
wrangler.toml       — Cloudflare Workers config
```
